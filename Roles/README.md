# Roles (角色) 说明

“角色”类型的提示词旨在定义 AI 的核心身份、专业领域和主要任务。

通过为 AI 分配一个明确的“角色”，我们可以引导它在特定的知识框架和行为模式下进行思考和回应，从而获得更专业、更可靠的输出。

## 可用角色列表

下表列出了当前所有可用的“角色”系列。点击链接可以查看每个系列下不同版本和变体的详细说明。

| :--- | :--- |
| [**`CodingAssistant`**](./CodingAssistant/README.md) | 提供专家级的编程、调试和系统设计辅助。 |
